Copy both lines of code and paste it in the head of your html:

<link rel="stylesheet" href="https://light-dark-theme.awesomearjun1.repl.co/style.css" />
<script defer src="https://light-dark-theme.awesomearjun1.repl.co/script.js"></script>

To use this, make a button in your html for changing the theme and in that html file and put an attribute like this in it:
onclick="changeTheme()"
